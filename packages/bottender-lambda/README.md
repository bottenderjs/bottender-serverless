# bottender-lambda

## Installation

```sh
npm install bottender-lambda
```

or

```sh
yarn add bottender-lambda
```

<br />

## Usage

```js
const { createRequestHandler } = require('bottender-lambda');

// ...

exports.handler = createRequestHandler(bot);
```
