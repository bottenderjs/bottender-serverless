# bottender-azure-functions

## Installation

```sh
npm install bottender-azure-functions
```

or

```sh
yarn add bottender-azure-functions
```

<br />

## Usage

```js
const { createRequestHandler } = require('bottender-azure-functions');

// ...

exports.handler = createRequestHandler(bot);
```
